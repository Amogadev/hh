# **App Name**: HotelView

## Core Features:

- Login/Authentication: Secure user authentication to access the dashboard.
- Dashboard Overview: Display key hotel statistics, including total rooms, booked rooms, and occupied rooms.
- Room Status Display: Show the status of rooms 101-106.
- Payment Details Section: Section to display payment-related information and transaction history.
- Firebase Integration: Use Firestore as a real-time database to manage room details, user data, and payments.
- API Key Generation: Automatically generates a Gemini API key using a tool, allowing use of the API.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust and reliability.
- Background color: Very light grayish-blue (#F0F4F9).
- Accent color: Muted violet (#8E24AA) for highlighting interactive elements and key information.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, and readable interface. 
- Use simple, clear icons to represent room statuses, payment methods, and other key features. Icons should be consistent in style and size.
- A clean, card-based layout that makes the most important information very easy to scan, on both desktop and mobile.
- Subtle transitions and animations to provide feedback to the user and improve the overall experience. For example, loading indicators or confirmation animations.
